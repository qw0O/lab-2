# Лабораторная работа №2
В данной программе для хранения списка остановок используется динамический массив (в языке Python представлен типом list).
---

###Описание созданных функций

В коде реализованы два класса: `BusStop` (данные об остановке) и `BusRoute` (маршрут автобуса).

#### Класс BusStop:
1.  **`__init__`** — инициализация объект остановки (название, координаты, время).
2.  **`to_string`** — формирует строку из данных остановки, подготовка к Base64.

#### Класс BusRoute:
1.  **`add_stop`** — создает новый объект остановки и добавляет его в динамический массив маршрута.
2.  **`calculate_total_time`** — проходит циклом по всем элементам массива и суммирует время до следующей остановки, вычисляя общую длительность пути.
3.  **`get_bus_after_n_stops`** — находит целевую остановку в массиве по индексу (текущий + $N$) и суммирует время в пути до этой точки.
4.  **`build_reverse_route`** — создает новый объект маршрута, проходя по исходному массиву в обратном порядке и пересчитывая временные интервалы.
5.  **`save_to_base64_file`** — кодирует текстовые данные каждой остановки в формат Base64 и записывает их построчно в файл.

Программа демонстрирует классическое использование динамических структур данных для моделирования реальных объектов (транспортных сетей). Сочетание высокой скорости чтения и гибкости добавления элементов делает список (`list`) оптимальным выбором для этой задачи.
